# esparto._publish

!!! info
    Publishing methods are accessed via Layout classes:

    ``` python
    import esparto as es

    # Create a new Page
    page = es.Page(title="My New Page")

    # Publish the document to an HTML file:
    page.save_html("my-page.html")

    # Or as a PDF:
    page.save_pdf("my-page.pdf")

    ```
