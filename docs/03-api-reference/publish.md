# esparto._publish

!!! info
    Publishing methods should be accessed via Layout classes.

    ``` python
    import esparto as es

    # Create a new Page
    page = es.Page(title="My New Page")

    # Render to Jupyter Notebook cell:
    page.display()

    # Publish the document to an HTML file:
    page.save_html()

    ```
